# QClaw Watchdog

QClaw 看门狗 - 监控并自动重启 QClaw 的工具

## 功能

- 自动监控 QClaw 状态
- 异常自动重启
- 飞书通知和指令控制

## 快速开始

```bash
git clone https://github.com/kf-liu/qclaw-watchdog-skill.git
cd qclaw-watchdog
./init-config.sh
./start.sh
```
